# 🎬 KachiFlix - Movie Discovery App

A modern movie discovery web app built with React and the OMDB API. Users can search for any movie, view details, ratings, and more.

## 🚀 Features
- Search movies in real time
- View movie details, ratings, and plot
- Responsive design for all devices
- Clean and modern UI

## 🛠️ Tech Stack
- React
- OMDB API
- CSS / Styled Components
- Vite

## 🔧 How to Run
1. Clone the repo
2. Run `npm install`
3. Run `npm run dev`
4. Open `http://localhost:5173`
